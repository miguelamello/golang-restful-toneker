Welcome to Tokener

Tokener aims to easy the creation of network wide tokens. It stands as a central service that can be used to create tokens that can be used by any other company's services across the company's network. Simply put, clients can create one unique token and use it across all of their intereactions with company's services.

The token is stored in a central database and it is indexed to the client's email. At anytime the client can request a new token by using the appropriate routes described on the service's Reference.
